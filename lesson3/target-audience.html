Target Audience

This website will be for anyone in the Idaho area that is intrested in knowing about weather and how to best plan around it for their outdoor plessures.

Site Personas

  1. Hikers
  2. Snowshoers
  3. Swimmers
  4. Photographers

Scenarios

  1. Checking to see if it will rain for a hike.
  2. Seeing if there is fresh snow to snowshoe on.
  3. Seeing if there will be any thunderstorms.
  4. Knowing when sunrise/sunest will be and cloud cover.
  5. Checking weather for road conditions for travel.
